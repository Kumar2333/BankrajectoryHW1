# BankTrajectoryHW1

This repository contains solutions for Homework 1.

## Files

### BankAccount.cpp
A simple BankAccount class demonstrating:
- Private data member (balance)
- Constructor
- Deposit function
- Withdraw function
- GetBalance function

### TrajectorySimulator.cpp
A projectile motion simulator that calculates:
- Time of flight
- Maximum height
- Horizontal range
- Vertical position at a given x value

The program uses standard kinematic equations and validates negative speed input.
